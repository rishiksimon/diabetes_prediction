# 🩺 Diabetes Prediction System

## 📌 Overview
This project predicts whether a patient has diabetes based on medical attributes using Machine Learning classification models.

## 📊 Dataset
Pima Indians Diabetes Dataset

## ⚙️ Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Streamlit (for UI)

## 🧠 Models Used
- Logistic Regression
- Random Forest
- Decision Tree

## 🚀 How to Run

1. Clone the repository
2. Install dependencies:
   pip install -r requirements.txt
3. Run:
   streamlit run app.py
