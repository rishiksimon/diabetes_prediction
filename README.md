# 🩺 Diabetes Prediction App

## 📖 Overview
The **Diabetes Prediction App** is a machine learning project that predicts the likelihood of diabetes in patients based on health-related attributes. It uses supervised learning algorithms to provide accurate predictions, helping in early diagnosis and preventive healthcare.

---

## ✨ Features
- Predicts diabetes risk using patient medical data.
- Built with **Python** and **scikit-learn**.
- Supports both **classification** and **regression** approaches.
- Data visualization for better insights.
- Modular code structure for easy extension.

---

## 🛠️ Tech Stack
- **Programming Language**: Python  
- **Libraries**:
  - `pandas` – data handling
  - `numpy` – numerical computations
  - `matplotlib` / `seaborn` – visualization
  - `scikit-learn` – machine learning models

---

## 📊 Dataset
- Uses the **Pima Indians Diabetes Dataset** (UCI ML Repository / Kaggle).
- **Features**:
  - Pregnancies
  - Glucose
  - Blood Pressure
  - Skin Thickness
  - Insulin
  - BMI
  - Diabetes Pedigree Function
  - Age
- **Target**: `Outcome` (1 = diabetic, 0 = non-diabetic)
